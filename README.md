# README

Get Started:

git clone

add the following gems:
<!-- for importing the csv files -->
    gem 'csv', '~> 3.0'

<!-- for testing  -->
    gem 'rails-controller-testing'
    gem 'minitest'  
    gem 'minitest-reporters'

<!-- rake task to import data from csv files to database -->
run rails update:import_all
